# React ToDo-app with a Firebase backend

To install the dependencies, use `npm install`
To run the project locally, use `npm start`

### Dependencies: 
- @testing-library/jest-dom
- @testing-library/react
- @testing-library/user-event
- firebase
- firebaseui
- react
- react-dom
- react-icons
- react-scripts
- web-vitals
